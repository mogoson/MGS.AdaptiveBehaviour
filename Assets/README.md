# MGS.AdaptiveBehaviour

## Summary

Adaptive MonoBehaviour for Unity project.

## Ability

- Adaptive transform child to target, face to target, hook to target.
- Adaptive transform scale base distance to target.
- Adaptive LineRenderer size base distance to target.

## Install

- Unity --> Window --> Package Manager --> "+" --> Add package from git URL...

  ```text
  https://github.com/mogoson/MGS.AdaptiveBehaviour.git?path=/Assets
  ```
## Samples

- Unity --> Window --> Package Manager --> Packages-Mogoson --> AdaptiveBehaviour --> Samples.
